# 封神榜 - 伐商演义
## 游戏策划案 v2.0

---

## 一、项目概述

| 项目 | 内容 |
|------|------|
| **项目名称** | 封神榜 - 伐商演义 |
| **类型** | 策略卡牌 RPG + 推图 |
| **平台** | 微信小程序 / Web（H5） |
| **核心玩法** | 玩家扮演阐教弟子，招募神仙、收集法宝、伐商封神 |
| **盈利模式** | 内购（月卡、首充）+ 广告变现 |

---

## 二、目标用户

- **年龄**: 18-40 岁
- **偏好**: 中国神话、国风仙侠、策略养成
- **特征**: 碎片化时间多、喜欢社交竞技

---

## 三、核心玩法循环

```
推图通关 → 获取资源（灵气、法宝碎片） → 升级伙伴/法宝 → 推更难关卡
     ↓                                            ↓
  每日任务 ←←←←←←←←←←←←←←←←←←←←←←←←←←←← 社交/竞技
```

---

## 四、游戏背景

商朝末年，纣王无道，宠信妲己，残害忠良，天下大乱。元始天尊命姜子牙下山伐商，辅佐周文王、武王。玩家作为阐教弟子，需招募各路神仙、获取法宝、经历封神大战，最终完成封神大业。

---

## 五、系统架构

### 5.1 伙伴系统 ⭐⭐⭐

| 元素 | 说明 |
|------|------|
| **伙伴来源** | 抽卡、推图解锁、活动赠送 |
| **阵营** | 阐教（玩家阵营）、截教（敌方/可招募）、商朝（敌方）、周（友方） |
| **品质** | 蓝 → 紫 → 橙 → 红（传说） |
| **星级** | 1-5 星，升星需要同名碎片 |
| **等级** | 跟随主角等级上限 |

**初始伙伴**：
- 哪吒（橙色）
- 杨戬（橙色）
- 雷震子（紫色）

**可招募伙伴**：
- 姜子牙、申公豹、金吒、木吒、韦护、李靖
- 土行孙、惧留孙、太乙真人、玉鼎真人
- 商朝：纣王、妲己、申公豹（截教）
- 赵公明、三霄仙子（截教）

### 5.2 战斗系统 ⭐⭐⭐

| 机制 | 说明 |
|------|------|
| **战斗模式** | 回合制自动战斗，玩家选择技能释放时机 |
| **上阵数量** | 5 个伙伴 + 主角 |
| **技能类型** | 普通攻击、主动技能、被动技能、终结技 |
| **能量系统** | 技能释放消耗能量，能量随回合累积 |
| **属性克制** | 火 → 木 → 土 → 水 → 金 → 火 |
| **法术克制** | 法术伤害 > 物理伤害 |

**战斗流程**：
1. 双方按速度排序先后手
2. 每回合开始获得能量
3. 玩家选择释放技能或普攻
4. 结算伤害，循环至一方全灭

### 5.3 法宝系统 ⭐⭐⭐

| 元素 | 说明 |
|------|------|
| **品质** | 蓝 → 紫 → 橙 → 红 |
| **强化** | 消耗同名法宝或材料升级 |
| **套装** | 2件/4件激活套装效果 |

**法宝列表**：
- 乾坤圈、风火轮、混天绫（哪吒套装）
- 打神鞭、杏黄旗、封神榜（姜子牙套装）
- 九龙神火罩、金砖、阴阳剑
- 落宝金钱、定海珠、混元金斗

### 5.4 推图系统 ⭐⭐⭐

| 元素 | 说明 |
|------|------|
| **章节** | 12 章（陈塘关→朝歌→牧野等） |
| **关卡** | 每章 10 关，共 120 关 |
| **难度** | 简单/普通/困难/精英 |
| **扫荡** | 通关后可直接扫荡 |
| **星级** | 每关 3 星（回合数、伙伴存活数、通关时间） |

**章节列表**：
1. 序章：陈塘关（哪吒闹海）
2. 第二章：渭水河（姜子牙下山）
3. 第三章：西岐（文王访贤）
4. 第四章：佳梦关（魔家四将）
5. 第五章：青龙关（哼哈二将）
6. 第六章：汜水关（余化）
7. 第七章：界牌关（诛仙阵）
8. 第八章：穿云关
9. 第九章：潼关（钉头七箭书）
10. 第十章：临潼关
11. 第十一章：渑池县
12. 第十二章：朝歌（牧野之战+封神）

### 5.5 资源系统 ⭐⭐

| 资源 | 用途 | 获取方式 |
|------|------|----------|
| **灵气** | 升级、突破 | 推图、每日任务 |
| **金币** | 强化、升星 | 推图、商城 |
| **经验** | 伙伴升级 | 推图 |
| **碎片** | 伙伴/法宝升星 | 推图、抽卡 |
| **贡献** | 联盟商店 | 联盟活动 |

### 5.6 城建系统 ⭐⭐

| 建筑 | 功能 |
|------|------|
| **聚灵阵** | 自动产出灵气 |
| **天材房** | 产出升级材料 |
| **论道台** | 伙伴经验加成 |
| **封神台** | 解锁新伙伴槽位 |
| **仙炉** | 法宝强化 |

### 5.7 社交系统 ⭐⭐

| 功能 | 说明 |
|------|------|
| **排行榜** | 战力榜、关卡榜、地区榜 |
| **好友** | 互赠体力、助力 |
| **联盟** | 加入阐教联盟，互助、联盟战 |
| **地区** | 按省份/城市显示地区排行 |

### 5.8 每日/成就系统 ⭐⭐

| 类型 | 内容 |
|------|------|
| **每日签到** | 连续登录奖励（灵气、金币、碎片） |
| **每日任务** | 推图 3 次、抽卡 1 次等 |
| **成就** | 通关章节、收集伙伴、累计充值 |
| **首充** | 首充 6 元送哪吒 |
| **月卡** | 每日领取 100 灵气 |

### 5.9 PVP 竞技 ⭐

| 模式 | 说明 |
|------|------|
| **论道** | 自动对战，玩家防守阵容 |
| **天梯** | 实时匹配，段位制 |
| **跨服** | 全服玩家对战 |
